Минимальные требования:

1. Написать не менее 5 тестов
Покрыть сценарии:

* добавление задачи
* удаление задачи
* добавление нескольких задач
* проверка пустой задачи
* проверка текста задачи

2. Используйте:
* pytest
* fixtures
* Selenium

3. Найти минимум 2 бага
4. Сформировать краткий отчёт

```BUG 1
Какой то баг

Шаги:
1 открыть приложение
2 Сделать действие

Фактический результат:
Баг проявлется

Ожидаемый:
Баг не проявляется```
